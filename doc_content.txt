Industrial Monitoring Scenario (Small-Scale Food Processing Industry, Dharavi, Mumbai) Industry Description The proposed system is deployed in a small-scale food processing and packaging industry located in Dharavi, Mumbai . The industry consists of: Raw material storage area Food processing section Packaging and dispatch section Outdoor loading and unloading yard The factory premises extend over approximately 1.5 km communication coverage area , which is suitable for evaluating the maximum practical range of a 433 MHz LoRa -based single-gateway architecture . Factory Layout Area Distance from Gateway Environment Packaging Unit 250 m Indoor Processing Unit 800 m Indoor Loading Yard 1500 m Outdoor Gateway Location The gateway is installed in the administrative office at the center of the factory and consists of: ESP32 LoRa SX1278 (433 MHz) 5 dBi Antenna Wi-Fi Module Dashboard Server Connection Sensor Node Configuration Each sensor node contains: ESP32 SX1278 LoRa Module (433 MHz) MQ135 Gas Sensor MQ2 Smoke Sensor MPU6050 Vibration Sensor HC-SR04 Ultrasonic Sensor Power Supply Unit The nodes continuously monitor: Gas leakage Smoke concentration Machine vibration Distance and obstruction conditions and periodically transmit data to the gateway.
